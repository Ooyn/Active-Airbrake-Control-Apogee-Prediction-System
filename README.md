# Active-Airbrake-Control-Apogee-Prediction-System
A MATLAB/Simulink 1-DOF flight simulator and PID control system for high-power rockets. Features a real-time apogee predictor using closed-form drag equations, dynamic aerodynamic lookup tables from OpenRocket, and active airbrake control to accurately target a 10,000 ft apogee with a strict 2000m deployment safety threshold.
